How to Play Simon

Simon is a memory game where you need to repeat a sequence of colors and sounds.

Game Rules:

  • Watch and listen to the sequence played by Simon
  • Repeat the sequence by clicking the colored buttons in the same order
  • Each successful round adds one more step to the sequence
  • If you make a mistake, the game ends and your score is saved

Difficulty Levels:

  • Easy: Slower sequence, longer flash duration (Max Level: 20)
  • Medium: Moderate speed and flash duration (Max Level: 30)
  • Hard: Fast sequence, short flash duration (Max Level: 50)

Try to achieve the highest level possible and compete for the top 5 scores in each difficulty!
